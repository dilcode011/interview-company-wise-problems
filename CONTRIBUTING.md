# Contributing to LeetCode Company-wise Problems

Thank you for your interest in contributing! This repository is community-driven and any help is welcome.

## Ways to Contribute

- **Update existing company data** — Refresh the CSV files with the latest LeetCode problems (especially useful since the last update was June 2025).
- **Add a new company** — Create a new folder following the exact same CSV format and file naming.
- **Improve documentation** — Enhance README, fix typos, add examples.
- **Bug fixes** — Correct wrong links, frequencies, topics, etc.
- **Scripts & automation** — Add helpful scripts (e.g., scraper) while respecting LeetCode’s Terms of Service.

**Note**: Accurate frequency counts often require LeetCode Premium. You can still contribute using public company tags via browser extensions.

## Pull Request Process

1. Fork the repo and clone your fork.
2. Create a feature branch (`git checkout -b feature/amazing-change`).
3. Make your changes.
4. Commit with a clear message.
5. Push to your branch and open a Pull Request.

Please include in your PR description:
- What you changed (e.g., “Updated Amazon CSVs as of March 2026”)
- Source of data (LeetCode company tags)

Thank you! Your contribution will help thousands of job seekers.
